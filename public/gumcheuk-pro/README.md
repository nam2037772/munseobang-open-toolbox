# 검측프로

문서방 오픈 툴박스의 품질관리용 독립 웹앱입니다. 건설현장의 검측요청서, 검측점검표, 검측결과통보서, 공사참여자실명부, 사진대지를 브라우저에서 작성하고 JSON/PDF로 보관합니다.

## 핵심 원칙

- HTML/CSS/JavaScript만 사용하는 완전 독립 실행 앱입니다.
- 빌드 과정, 서버, 데이터베이스, 외부 CDN 의존성이 없습니다.
- GitHub Pages와 Vercel 정적 배포에서 그대로 실행됩니다.
- 서버 저장을 하지 않으며, 자동 임시저장은 브라우저 `localStorage`만 사용합니다.
- 장기 보관은 사용자가 내려받는 JSON/PDF 파일로 수행합니다.
- 검측 결과와 최종 판정은 사용자가 직접 선택합니다.

## 설치 방법

별도 설치가 필요 없습니다. 이 폴더 전체를 그대로 보관하거나 정적 호스팅에 업로드하면 됩니다.

```
검측프로/
├── index.html
├── style.css
├── inspectionTemplates.js
├── app.js
└── README.md
```

## 실행 방법

가장 간단한 실행은 `index.html`을 브라우저로 여는 방식입니다. 파일 선택/다운로드 동작을 더 안정적으로 확인하려면 로컬 정적 서버를 권장합니다.

```bash
cd "검측프로"
python -m http.server 8080
```

브라우저에서 `http://localhost:8080`으로 접속합니다.

## 주요 기능

- 현장 기본정보 입력 및 미리보기 자동 반영
- 공종/세부공종 선택과 기본 검사항목 자동 생성
- 검사항목별 시공자 1차/2차, 감리자 1차/2차 결과 직접 선택
- 검사항목 추가/삭제, 조치사항/특기사항 입력
- 캔버스 서명 입력 및 문서 반영
- 사진 여러 장 첨부, 자동 리사이즈, 설명 입력, 검사항목 연결, 삭제
- 검측결과통보서 작성 및 감리 서명
- 공사 참여자 실명부 작성
- JSON 저장/불러오기
- `localStorage` 자동 임시저장 및 복원
- 문서별 PDF 출력과 전체 묶음 PDF 출력

## 포함된 공종 템플릿

`inspectionTemplates.js`에 대공종과 세부공종 템플릿이 분리되어 있습니다. 현재 기본 템플릿은 일반적인 건설 검측 실무 기준으로 구성되어 있으며, 현장 시방서와 사내 ITP 기준에 맞게 조정해서 사용하는 것을 전제로 합니다.

포함 예시:

- 토공사: 터파기
- 철근공사: 기초철근배근, 벽체/기둥 철근배근
- 거푸집공사: Slab 배근 및 거푸집공사
- 콘크리트공사: 버림콘크리트, 타설 전/중/후
- 방수공사: 바탕면 처리, 시멘트 액체방수
- 조적, 미장, 타일, AL창호, 도장, 철골, 금속, 석공사, 기타 마감공사 공통 템플릿

## 향후 확장 방법

새 공종이나 세부공종은 `inspectionTemplates.js`만 수정하면 됩니다.

1. 대공종이 없으면 `WORK_CATEGORIES`에 `{ id, name }`을 추가합니다.
2. `SUB_WORK_TEMPLATES`에 `{ code, name, category, items }` 객체를 추가합니다.
3. `category`는 `WORK_CATEGORIES.id`와 일치해야 합니다.
4. `items`는 `{ item: "검사항목", standard: "검사기준" }` 배열로 작성합니다.

`app.js`는 이 데이터 구조만 읽으므로, 보통 앱 로직 수정 없이 선택박스와 검사항목 자동 생성에 반영됩니다.

## 알려진 제한사항

- 원본 HWP 검측업무지침과 100% 대조한 데이터가 아닙니다. 실제 배포 전 현장 시방서, 감리단 기준, 사내 ITP와 대조해야 합니다.
- 사진은 브라우저 안에서 리사이즈하지만, 많은 사진을 넣으면 `localStorage` 용량 한도 때문에 자동 임시저장이 실패할 수 있습니다. 이 경우 JSON 저장을 우선 사용하세요.
- PDF는 브라우저 인쇄 기능을 사용합니다. 브라우저/프린터 설정에 따라 여백과 배경색 출력이 달라질 수 있습니다.
- 다중 검측 건 목록 관리는 제공하지 않습니다. 건별 JSON 파일로 저장하고 다시 불러오는 방식입니다.
- 서명은 이미지 데이터로 저장됩니다. 별도 전자서명 인증, 위변조 방지, 서버 타임스탬프 기능은 없습니다.

## 배포 전 점검

- 불필요한 임시 파일이 없는지 확인
- 브라우저 콘솔 오류 확인
- JSON 저장/불러오기 확인
- 사진 첨부 후 자동저장 실패 여부 확인
- Chrome 인쇄 대화상자에서 문서별 PDF와 전체 묶음 PDF 확인
- 모바일/태블릿에서 버튼 크기와 서명 입력 확인
