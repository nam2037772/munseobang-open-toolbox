# 변경 내역

## 0.1.1 - 2026-07-05

### 문서

- `MANIFESTO.md`에 문서 우선 원칙을 추가했다.
- `PROJECT_LOG.md`에 오늘 구현한 기능, 결정한 정책, 수정한 내용, 발견한 문제, 다음 작업을 정리했다.
- `ROADMAP.md`에 완료 항목을 체크하고 다음 개발 항목을 보강했다.
- 프로젝트 문서를 새로운 AI와 개발자가 먼저 읽어야 하는 기준 문서로 정리했다.

## 0.1.0 - 2026-07-05

### 변경

- 문서방 MVP 첫 화면의 한글 문구를 복구했다.
- 검색을 프론트엔드 정적 데이터 필터링으로 정리했다.
- 검색 결과 카드를 제목, 분류, 한 줄 답, 설명, 신뢰 라벨, 제공 기능, 열기 버튼 구조로 정리했다.
- 회원가입 없음, 서버 저장 없음, 공식자료 우선, 내 PC 저장 원칙을 화면에 반영했다.
- 모바일에서도 카드가 한 열로 안정적으로 보이도록 CSS를 정리했다.
- GitHub Pages 하위 경로 배포를 고려해 Vite 상대 경로 설정을 반영했다.

### 추가

- `MANIFESTO.md`
- `PROJECT_LOG.md`
- `ROADMAP.md`
- `CHANGELOG.md`

### 검증

- `npm.cmd install` 성공
- `npm.cmd run build` 성공

### 제거하지 않음

- 로그인, DB, 서버 저장, 게시판 기능은 애초에 추가하지 않았다.
