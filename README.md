# 문서방

**건설인 오픈 툴박스**

문서방은 커뮤니티가 아닙니다. 자료창고도 아니고, AI 검색 사이트도 아닙니다.

문서방은 건설인이 검증된 답과 도구를 바로 꺼내 쓰고, 결과물을 자기 PC에 저장하는 신뢰 기반 오픈 툴박스입니다.

> 우리는 정보를 모으지 않는다. 우리는 답을 정리한다.

## 핵심 원칙

1. 회원가입을 요구하지 않습니다.
2. 사용자 문서와 현장자료를 서버에 저장하지 않습니다.
3. 정부 공식자료, 법령, KS, KCS, KDS, 표준시장단가 등 공신력 있는 자료를 우선합니다.
4. 검색 결과를 많이 보여주기보다 검증된 답안 카드로 빠르게 이동하게 합니다.
5. 커뮤니티와 토론은 기존 카페가 합니다. 문서방은 기준과 도구를 제공합니다.
6. AI는 전면 기능이 아니라, 향후 필요한 곳에서만 보조 도구로 다룹니다.

## 기술 스택

- Vite
- React
- TypeScript
- 일반 CSS
- 정적 프론트엔드 데이터: `src/data/tools.ts`

현재 MVP에는 로그인, DB, 서버 저장, 파일 업로드 보관 기능이 없습니다.

## 실행 방법

```bash
npm install
npm run dev
```

프로덕션 빌드:

```bash
npm run build
npm run preview
```

빌드 결과물은 `dist/`에 생성되며 GitHub Pages 또는 Cloudflare Pages에 정적 파일로 배포할 수 있습니다.

## 현재 MVP 범위

- 첫 화면 브랜드/원칙 안내
- 프론트엔드 내부 데이터 기반 검색
- 검증된 답안 카드 UI
- 도구/기준 카테고리 안내
- 하지 않을 것과 저장 정책 명시
